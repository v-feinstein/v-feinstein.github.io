Rated!!


Intro

Rated!! is an app that helps developers to monitor user ratings of their published apps.

The app periodically polls Apple site for updated ratings of each app in the configured countries.

To begin, you need to add the apps you would like to monitor by tapping “Add apps to monitor” button or through the “Apps” menu.

The free Basic edition allows you to monitor one app in up to three countries.

The Premium edition lets you monitor up to three apps in unlimited number of countries.

The Pro edition ads an unlimited number of apps to that.

Add Apps

To add an app to monitor the user ratings for, you need to know either that app ID or the developer ID.

You can type either one in the edot box at the top of the screen and tap the “Lookup” button.

If you entered an app ID, that app will be added to the list below.

Add Apps by Developer’s ID

If you entered a developer’s ID, Rated!! will look up all the apps published by that developer and add them to the list below.

At this point, you can select any or all apps and tap “Add selected” button.

Please note that you do not need any credentials to look up the ratings of an app. It doesn’t even have to be your app.

Main View

Now the apps are loaded into the main view.

For each app, the following information is displayed:

  • the app’s icon
  • the name of the app
  • the latest released version
  • the date of the initial release
  • number of user ratings
  • an average user rating for that app

Please note that the new ratings received after the last time you looked at that app’s details will be color-coded: green if the average rating went up, red if it went down. The badge on the app’s icon indicates how many new ratings were received.

Gestures

You can control the app with the following gestures:

  • Tap an app to see detailed information
  • Pull down to refresh the data
  • Swipe left to remove an app from monitoring

Menu

You can perform the following action through the menu:

  • Refresh - query current ratings of all apps
  • Mark All as Read - drop the badges and color coding from all apps
  • Apps - add new apps to monitor
  • Countries - configure the list of countries to monitor
  • How To Use - see brief documentation
  • Upgrade - upgrade an app to Premium or Pro edition
  • About - brief information about this app, including the links to our website and our Facebook page
  • Journal - see records of manual and background updates

Countries

By default, only the United States will be monitored.

If you would like to monitor additional countries, please go to the Countries page through the menu.

There you would see five regions, as defined by Apple, with the lists of countries in each:

  • The United States and Canada
  • Europe
  • Africa, Middle East, and India
  • Latin America and the Caribbean
  • Asia Pacific

You can tap the chevrons to expand or collapse each region. You can also tap the toggle by the region name to select or unselect all countries in that region.

Please note that the free Basic edition allows you to monitor up to three countries. Please upgrade to Premium or Pro to monitor unlimited number of countries.

App View

When you tap on an app in the main view, you will see the details for that app:

  • App’s name
  • Latest released version
  • Original release date
  • Release date of the current version
  • Count of user ratings received
  • Average user rating

Below that, you will see a list of all monitored countries where that app received any ratings.

You can sort that list by tapping either the Country or the Rating heading.

The number of ratings and the average rating are color-coded indicating the change since you last looked at this app:

  • green with the arrow up - the average rating moved up
  • red with the arrow down - the average rating moved down
