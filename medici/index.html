Medici


Rules

Medici Solitaire is played with a deck of 36 cards.

First, deal three cards from the deck. If a middle card is between the cards of the same suit (e.g., hearts) or between cards of the same rank (e.g., tens), then this card is shifted on top of the previous one. Deal another card by taping on the deck, repeat, and so on.

Solitaire converged when, after all possible moves, only two cards are left on the table.

Menu

You can control the game through the menu with the following commands:

  • New Game - reshuffle the deck and start a new play.
  • Undo - undo, one at a time, your moves
  • Start Over - start a new play with the same deck

  • Show Hint - if any cards on the table can be moved, highlight the first one of them from the left. Also display a message at the bottom of the screen.

  • Show Warning - similar but less intrusive feature. If you move a card while there was another valid move on the left, it will display a warning at the bottom of the screen. At this point, you should use the Undo command and look for that move. You can trigger a Show Hint option if you can’t find it.
  • Deal Entire Deck - place all the cards on the table so that you don’t have to tap the deck for each new card.
  • Auto Play - let the AI play for you. At the end, if you like how it ended, you can use Start Over and play the same deck yourself.
  • Leaderboard - we record the date and time when you have this solitaire converge. You can look back at those events.
  • About - brief information about this app, including the links to our website, this page and our Facebook page.

Game Over

This solitaire ends when there are no more valid moves on the table and the deck is empty. At this point you can start the new game or repeat with the same deck, in case you missed the move.

Congrats

In the rare event that this Medici Solitaire converged, you will see the message on the left.

We will record this event in the leaderboard.
