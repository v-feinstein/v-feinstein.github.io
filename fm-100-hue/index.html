Farnsworth–Munsell 100 Hue Color Vision test


Background

Farnsworth-Munsell 100 is a color vision test often used to test for color blindness.

This mobile adaptation of the test contains four distinct rows of similar color hue tiles, each containing 12 distinct variations of each hue. The tiles at the ends of each row are fixed in position, to serve as anchors.

Please see Wikipedia for more details.

How to use

To start the test, please tap Shuffle button. The tiles will be shuffled within each row.

Then you can drag each tile, within its row, to arrange them in the original order. You can also tap the tile you want to move, then tap the arrows on either end of the row to move it left or right.

When you finish, please tap Done button to see the result.

Result

The test results are presented in a Polar Chart that indicates how far from the target the color tiles were placed.

Disclaimer: this app is not intended to diagnose your color vision deficiency, but simply to indicate if you should follow up on that matter.
