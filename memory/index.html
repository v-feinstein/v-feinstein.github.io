Memory


Rules

Memory is a classic card game, also known as Concentration.

The cards are dealt from the double deck into a player-selected tableau face down.

A player chooses (taps) two cards that are turned face up. If they match (the same rank and suit), they are removed from the table. Otherwise they are turned face down, and the game continues until there are no more card on the table.

Scoring

When the player uncovers a mismatched pair, they get one “Oops”.

The object of the game is to clear the tableau in the fewest number of turns, or to get the lowest possible score.

Memory is a very challenging game, especially on large tableaus. To make it more competitive (with yourself), we keep a score for you, separate for each tableau size.

Menu

You can control the game through the menu with the following commands:

  • New Game - reshuffle the deck and start a new play.
  • Settings - configure different options the way you like it.
  • Scoreboard - see your best scores, for each tableau size.
  • How To Play - see the documentation.
  • About - brief information about this app, including the links to this website and our Facebook page.

Variations

This Memory game is appropriate for the players of all ages.

However, for our youngest players we provide a special option with the letter cards instead of the standard playing cards, so that they can enjoy the game before they learn the playing cards basics.

Settings

You can configure different options the way you like it:

  • Playing cards - - use the standard deck of playing cards.
  • Card size - select small, medium or large print for the cards.
  • Letters - use letter cards instead.
  • Tableau size - choose from the multiple tableau sizes, making the game more and more challenging.
  • Show all cards on start - show all the cards after they are dealt, until you tap any of them. This makes it a little easier to start the game on a large tableau.

Scoreboard

To help you track your progress, we keep records of your best scores with the date when they were achieved, separate for each tableau size.
