Alaska Solitaire


Rules

Alaska Solitaire is played with one deck of 52 cards.

The cards are dealt in 7 rows of tableau stacks. Their lengths, from left to right, are: 1, 6, 7, 8, 9, 10, 11. The topmost 5 cards of each stack are face-up, while the rest of the cards underneath them are face-down.

There are 4 foundations that build up from Aces, in suit.

Groups of cards can be moved; the cards below the one to be moved do not need to be in any order, except that the starting and target cards must be built in either up or down sequence and in suit. For example, a group starting with a 5♠ (or 7♠) can be moved on top a 6♠, and the cards below the 5♠ can differ.

When a face-down card becomes topmost, it is turned face up.

An empty tableau spots can be filled with any card or any group of cards.

The game is won when all the cards are in the foundations with the Kings on top.

Variants: strict mode allows only Kings or a group of cards starting with a King to be moved to an empty tableau spot.

Menu

You can control the game through the menu with the following commands:

  • New Game - reshuffle the decks and start a new play.
  • Hint - provide an on-screen hint for all valid moves. The cards that can be moved are outlined in green, the locations where they can be moved to - in yellow. If a card can be moved and another card can be moved on it, it will be outlined in orange.
  • Explain - show and explain all possible moves.
  • Settings - configure different options the way you like it.
  • Scoreboard - we record the number of times you have this solitaire converged. You can get to the Game Center Leaderboard to compare your score to other players.
  • How To Play - see the documentation.
  • About - brief information about this app, including the links to our website and our Facebook page.

 

Gestures

You can control the game with the following gestures:

  • Tap a card on the table to send it to the appropriate place in the foundation or another row in the tableau if there is a valid move.
  • Drag a card (or a group of cards) to where you want to move it to.
  • Swipe right - show Hint.
  • Swipe down - show and explain all possible moves.
  • Swipe up - bring up the Settings view.
